# Java To‑Do List Application

## 📌 Project Description
This is a simple **console-based To-Do List application** written in Java.  
It allows users to:
- ✅ Add new tasks
- 📋 View all tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks

It's a basic CRUD (Create, Read, Update, Delete) project built using Core Java concepts.

---

## 🛠️ Technologies Used
- Java (Core)
- ArrayList for storing tasks
- Console-based input/output (Scanner class)

---

## 🧠 Java Concepts Demonstrated
- Object-Oriented Programming (OOP)
- Java Classes & Methods
- Collections (ArrayList)
- Conditional Logic and Loops
- Exception Handling

---

## 🚀 How to Run the Project

### 💻 Requirements
- Java installed (JDK 8 or above)
- Any code editor (VS Code, IntelliJ, Eclipse, or simple Notepad)

### 🧾 Steps
1. Clone this repository:
git clone https://github.com/SadhuAkshaya/TODO_LIST.git
2. Navigate to the project directory.
3. Compile the code:
javac Main.java
4. Run the program:
java Main
